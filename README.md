# EasyOpAmp
Breakout board for OpAmp 

EasyOpAmpボードはsoic8の2回路入りオペアンプを2.54ピッチに変換するブレイクアウトボードです。  
ボード上にオペアンプの図が書かれているため、試作や実験の際に誤配線防止やデータシートの参照が不要になります。  
なお、ブレッドボード上の配線を綺麗にするため、一般的なオペアンプのピン配置と異なり、2つオペアンプが並列に並んだピン配置となっています。
